Involúcrate

Última Actualización: 4 nov. 2019

Comunidad

Siempre podemos ayudar contestando preguntas en Apoyo Comunitario Let’s Encrypt. Ve esta entrada de blog para ver por qué son importantes las contribuciones de apoyo comunitario.

Código

Siempre podemos usar ayuda para el desarrollo de software. Todo nuestro código esta en GitHub.

Software Cliente

Certbot es una utilidad basada en Python que trabaja junto a tu servidor web para obtener automáticamente un certificado y convertir un sitio web a HTTPS. Certbot es el cliente que recomendamos para comenzar para la mayoría de las personas. Muchas otras opciones de terciarios están disponibles.

Software AC del lado del servidor

Boulder es la implementación AC de Let’s Encrypt. Está basado en el protocolo ACME, y escrito primariamente en Go. Un gran lugar para comenzar es con la lista de problemas ‘help wanted’ y la guía de contribuyentes.
