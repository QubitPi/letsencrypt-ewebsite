帮助 Let's Encrypt

最后更新：Nov 4, 2019

社区论坛

您可以帮助我们在Let’s Encrypt 社区论坛中回答问题。 参见此博客文章以了解社区支持为何如此重要。

代码

您也可以帮助我们开发软件。 我们的所有代码都存储在 GitHub 上。

客户端软件

Certbot 是一款使用 Python 语言编写的实用程序，能够与服务器软件协同运作，自动获取证书并为网站部署 HTTPS 协议。 我们建议大多数用户都使用 Certbot 作为客户端， 但也有众多第三方客户端可供选择。

服务器端证书签发软件

Let’s Encrypt 的证书签发过程由 Boulder 实现。 该软件基于 ACME 协议，主要使用 Go 语言编写。 标注为“help wanted”的 issue 列表和贡献者指南都是一个很好的起点。
